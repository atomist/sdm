Class GoalSigningAutomationEventListener

AutomationEventListener that verifies incoming SDM goals against a set of configurable verification public keys.

Optionally a private key can be specified to sign outgoing goals. Setting this is strongly recommended to prevent executing untrusted and/or tampered SDM goals.
