  • Defined in lib/pack/aws/cache.ts:25

Type declaration

  • Optional bucket?: string

    AWS S3 bucket to perist cache entries to. If not provided, it defaults to "sdm-WORKSPACE_ID-SDM_NAME-goal-cache", with "WORKSPACE_ID" replaced with your Atomist workspace ID and "SDM_NAME" replaced with the name of the running SDM, converting letters to lower case and removing all characters that are not letters, numbers, and dashes (-). It makes no attempt to create this bucket, so make sure it exists before trying to use it.

  • Optional enabled?: boolean

    Set to true to enable goal input/output caching

  • Optional path?: string

    Path prefix, defaults to "goal-cache".

  • Optional region?: string

    AWS region for cache bucket.
